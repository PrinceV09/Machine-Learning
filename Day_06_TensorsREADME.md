# What are Tensors?
A Tensor is a generalized mathematical object used to store data in multiple dimensions.
A Tensor is an N-dimensional array of numbers.

| Name | Example | Dimension |
|------|---------|-----------|
| 0D / Scalar | `5` | 0D |
| 1D / Vector | `[1, 2, 3]` | 1D |
| 2D / Matrix | `[[1, 2], [3, 4]]` | 2D |
